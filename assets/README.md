# Assets

Put the college banner image here as **college-banner.png**
(the Sri Shakthi header with the NBA logo, college emblem, and name).

- Shown at the top of the dashboard banner (white strip, max height 96px, scales to fit).
- If the file is missing, the dashboard automatically falls back to the text heading.
- A `.jpg` also works if you update the filename in app.js (`assets/college-banner.png`).
